My First 3D Game!!

This is the first 3D game I ever made. I made it through following a Unity tutorial.The game is a simple roll a ball. All you have to do is collect all the squares and you win! Use WASD or the arrow keys to control the ball.

My First Game with Unity Standard Assets!

Feel free to have a look at my walking sim game made with with unity standard assets as part of my college work. The game has no end goal so do whatever you like. Use WASD and the arrow keys to control.

My Dark Souls 2 recreation!

This is a recreation of the forest of fallen joints starting area in dark souls 2. It is a simple walking simulator that shows my use of unity's terrain editor. It also shows some unity particle effects. WASD is used to move and spacebar is used to jump.

My biggest game so far!

This game is called Chernobyl. It is the biggest game I have made along side a friend. This was part of our college work. The game is set in chernobyl and it tells a story about a person investigating what really happened there. The object of the game is to collect all the bombs to destroy all the bridges in an attempt to keep the zombies away from the outside world. If you click the image it will bring you to a windows downloadable version of the game. Use WASD and the arrow keys to control.
